# MeningococcusFlow
Data pipeline to parse public data and map the spread of antibiotic-resistant and hyper-virulent strains of Neisseria Meningitidis

# Disclaimer 
All data belongs to pubmlst.org and NCBI

# Requirements
 - Python
 - Bash
 - Docker
 - Terraform
 - Google Cloud Storage
 - Google Big Query
